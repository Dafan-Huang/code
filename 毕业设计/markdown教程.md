# Markdown 教程

Markdown 是一种**轻量级标记语言**，非常适合编写文档、笔记和 README。  
它语法简洁、上手快，写出来的内容也清晰易读。

---

## 1. 标题

使用 `#` 表示标题，`#` 越多，标题级别越低。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

效果：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

---

## 2. 段落与换行

直接输入文字即可形成段落。

```markdown
这是第一段。

这是第二段。
```

如果想要**强制换行**，可以在行尾加两个空格，或使用 `<br>`。

```markdown
第一行  
第二行
```

---

## 3. 加粗、斜体、删除线

```markdown
**加粗文字**
*斜体文字*
~~删除线文字~~
```

效果：

**加粗文字**  
*斜体文字*  
~~删除线文字~~

---

## 4. 列表

### 无序列表

```markdown
- 苹果
- 香蕉
- 橙子
```

效果：

- 苹果
- 香蕉
- 橙子

### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

效果：

1. 第一项
2. 第二项
3. 第三项

### 嵌套列表

```markdown
- 学习内容
        - Markdown
        - Git
        - Python
```

---

## 5. 引用

使用 `>` 表示引用。

```markdown
> 这是一段引用。
```

效果：

> 这是一段引用。

---

## 6. 代码

### 行内代码

```markdown
`print("Hello, Markdown")`
```

效果：

`print("Hello, Markdown")`

### 代码块

使用三个反引号包裹代码，并在开头标注语言（可选）。

````markdown
```python
def hello():
        print("Hello, Markdown")
```
````

效果：

```python
def hello():
        print("Hello, Markdown")
```

## 8. 图片

```

```

效果：

- [ ] 学习代码块

## 12. 常用符号总结

| 功能 | 写法 |
|------|------|
| 标题 | `#` |
| 加粗 | `**文字**` |
| 斜体 | `*文字*` |
| 删除线 | `~~文字~~` |
| 引用 | `>` |
| 列表 | `-` 或 `1.` |
| 代码 | `` `代码` `` |
| 代码块 | ` ```语言` 与 ` ``` ` |
| 链接 | `[文字](地址)` |

---

## 13. 一个完整示例

````markdown
# 我的第一篇 Markdown

## 简介

这是我写的第一篇 Markdown 文档。

### 我喜欢的事情

- 写代码
- 看文档
- 学新知识

### 示例代码

```python
print("Hello, world!")
```

### 说明

> 学习 Markdown 可以让写文档更轻松。

更多内容请访问 [GitHub](https://github.com)。
````

---

## 14. 学习建议

1. 先掌握标题、列表、加粗、代码块。
2. 多练习写 README 和笔记。
3. 遇到不熟悉的语法时，优先查阅官方文档或高质量教程。
4. 最重要的是：边写边学。

---

## 15. 结语

Markdown 适合快速写作和整理内容。  
熟悉常用语法后，就能轻松写出结构清晰、排版整洁的文档。
