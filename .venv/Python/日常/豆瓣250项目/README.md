# 豆瓣电影Top250浏览器

一个基于Python和tkinter的豆瓣电影Top250浏览应用程序，支持电影信息浏览、海报展示和详情查看。

## 功能特点

- 📊 **电影排行榜**: 获取并显示豆瓣Top250电影排行榜
- 🖼️ **海报展示**: 支持电影海报的悬停预览和详情查看
- 🔍 **搜索功能**: 支持按电影名称、年份、评分等进行搜索
- 📱 **响应式界面**: 适配不同屏幕尺寸的现代化界面
- 🌐 **网络状态检测**: 内置网络连接测试和诊断功能
- 💾 **数据缓存**: 图片缓存和数据备份，提升用户体验
- 🔄 **错误处理**: 完善的错误处理和重试机制

## 安装和使用

### 环境要求

- Python 3.7+
- tkinter (通常随Python安装)
- 网络连接（用于获取豆瓣数据）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

## 主要模块说明

### 核心模块 (src/core/)

- **config.py**: 包含应用程序的所有配置信息
- **data_fetcher.py**: 负责从豆瓣网站获取电影数据
- **network.py**: 网络连接测试和诊断工具

### 用户界面模块 (src/ui/)

- **main_window.py**: 主窗口实现，包含电影列表和搜索功能
- **detail_window.py**: 电影详情窗口，显示详细信息和海报

### 工具模块 (src/utils/)

- **image.py**: 图片缓存和加载管理
- **helpers.py**: 通用工具函数和辅助类

## 使用说明

1. **启动程序**: 运行 `main.py` 启动应用程序
2. **浏览电影**: 主界面显示豆瓣Top250电影列表
3. **搜索电影**: 使用搜索框按名称、年份或评分筛选电影
4. **查看海报**: 鼠标悬停在电影名称上查看海报预览
5. **详情查看**: 双击电影名称或按回车键查看详细信息
6. **网络测试**: 点击"网络测试"按钮检查网络连接状态
7. **刷新数据**: 点击"刷新数据"按钮重新获取最新数据

## 快捷键

- `Ctrl+R` 或 `F5`: 刷新数据
- `Ctrl+F`: 聚焦到搜索框
- `Esc`: 清空搜索框或关闭详情窗口
- `Enter`: 查看选中电影的详情
- `双击`: 查看电影详情

## 故障排除

### 常见问题

1. **网络连接失败**
   - 检查网络连接
   - 使用"网络测试"功能诊断
   - 查看防火墙设置

2. **数据加载缓慢**
   - 检查网络速度
   - 尝试刷新数据
   - 使用备用数据模式

3. **界面显示异常**
   - 检查屏幕分辨率设置
   - 重启应用程序
   - 更新显卡驱动
